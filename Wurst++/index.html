Key Features

Wurst++ includes all of the original features from the official client, which you can see on the official Wurst website,
plus these unique additions:

  • Hack - AutoMLG -
  • No Analytics - Removes analytics of the official wurst client.
  • More smaller features...

Wurst++ is a fork of the Wurst Client and is not an official Wurst product.

The official Wurst Client can be found at wurstclient.net.
